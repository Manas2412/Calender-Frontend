# Aura Calendar | Premium Wall Calendar & Digital Planner

An interactive, high-fidelity calendar component that offers two distinct modes: a physical **Wall Calendar** aesthetic and a modern **macOS Digital Planner** view.

![Aura Calendar Showcase](public/hero.png)

## ✨ Pro Features

- **Dual View Modes**:
  - **Wall Month**: A beautiful, physical-inspired calendar with hero imagery and binding pins.
  - **Digital Day**: A 3-column macOS-inspired productive view with a source sidebar, hourly time grid, and inspector.
- **Smart Range Selection**:
  - Intuitive click-to-start and click-to-end range selection.
  - Real-time hover preview of the potential range.
- **macOS-Style Reminders & Scheduling**:
  - **Hourly Grid**: View scheduled tasks in a time-based layout.
  - **Daily Checklists**: Detailed task management for specific dates.
  - **Visual Badges**: System-native blue badges indicate days with active reminders.
- **Persistence**: All data (notes, reminders, view preferences) is persisted to `localStorage`.
- **Dual-Theme Support**: Native support for premium Light and macOS-fidelity Dark modes.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (CSS Modules)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🚀 Getting Started

1.  **Install dependencies**: `npm install`
2.  **Start development server**: `npm run dev`
3.  **Visit**: [http://localhost:3000](http://localhost:3000)
